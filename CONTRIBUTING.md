# Инструкция по настройке

## Настройка OAuth приложения

1. Перейдите на https://oauth.yandex.ru/client/new
2. Заполните форму:
   - **Название**: Умные лампочки (или любое другое)
   - **Платформа**: Веб-сервисы
   - **Redirect URI**: 
     - Для GitHub Pages: `https://egaliy.github.io/yandex-smart-lights/`
     - Для локальной разработки: `http://localhost:8000/`
3. В разделе **Доступ к данным** отметьте:
   - `iot:view`
   - `iot:control`
4. Сохраните **Client ID**

## Использование

1. Откройте веб-интерфейс
2. Введите ваш Client ID
3. Нажмите "Войти через Яндекс"
4. Авторизуйтесь и предоставьте права приложению
5. Готово! Вы будете перенаправлены обратно с токеном
